# Vite Vue3 Router Tailwind ESLint Prettier

Vite starter template that includes:

- Vue 3
- Vue Router
- Tailwind CSS
- ESLint
- Prettier
- @ import alias

## How to Use

```
npx degit danielkellyio/vite-vue3-router-tailwind-eslint-prettier [project-name]
```

```
npm install
npm run dev
```
